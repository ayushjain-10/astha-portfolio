## Getting Started

1. Simply open `index.html` in your browser
2. Or run a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

## Customization

### Adding Your Content

1. **Photos**: Replace the placeholder divs in `index.html`:
   - Hero section: Add your main photo
   - About section: Add 3 photos for the stack
   - Replace placeholder text/images throughout

2. **Videos**: Update the video section with your actual video embeds or links

3. **Information**: Update all text content with your actual:
   - Name, bio, and story
   - Contact email
   - Social media links
   - Class descriptions and pricing

### Styling

All colors and fonts are defined as CSS variables in `styles.css`:

```css
:root {
    --color-accent: #e85d75;      /* Main accent color */
    --color-secondary: #f4a261;   /* Secondary accent */
    --color-tertiary: #7b5ea7;    /* Tertiary accent */
    /* ... and more */
}
```
